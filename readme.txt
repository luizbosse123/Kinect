1) Instalar o Kinect SDK 1.8, diponível em https://www.microsoft.com/en-us/download/details.aspx?id=40278.
2) Instalar o Kinect for Windows Development Toolkit 1.8, disponível em https://www.microsoft.com/en-us/download/details.aspx?id=40276.

Para compilar o projeto, basta abrir o arquivo KinectMath.sln no Visual Studio 2013 ou superior e compilar.
As bibliotecas serão baixadas automaticamente via Nuget.

O núcleo da aplicação possui testes unitários implementados.
Para rodá-los, é necessário instalar no Visual Studio a extensão NUnit Test Adapter. Para isso, clicar em Tools -> Extensions and Updates...
